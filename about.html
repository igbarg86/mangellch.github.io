Sobre Mi

Soy una programadora fullstack apasionada por crear aplicaciones web robustas y eficientes. También fanática de los videojuegos retro y la tecnología. Con sólida experiencia en el desarrollo front-end y back-end, tengo un dominio completo de tecnologías clave que abarcan el ciclo de vida del desarrollo de software. Mi experiencia incluye:

  • Front-End: Experta en HTML, CSS y JavaScript, con habilidad para construir interfaces de usuario interactivas y responsivas. Utilizo React para desarrollar aplicaciones modernas y escalables, aplicando principios de diseño y mejores prácticas de desarrollo.
  • Back-End: Amplia experiencia en Node.js y Express para construir servidores eficientes y APIs RESTful. Manejo bases de datos SQL para gestionar y manipular datos de manera efectiva, asegurando integridad y rendimiento.

Mi enfoque está en crear soluciones elegantes y funcionales que ofrezcan una experiencia de usuario excepcional, y siempre busco aprender nuevas tecnologías y métodos para mejorar mis habilidades y los resultados de mis proyectos.
